# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/saturn/issues) to track bugs and feature
requests. Please search the existing issues before filing new ones to avoid duplicates. For new issues, file
your bug or feature request as a new Issue.

For help and questions about using this project, start with the [README](README.md) and the docs under
[`docs/`](docs/) — [get-started](docs/get-started.md), [how-saturn-works](docs/how-saturn-works.md), and
[architecture](docs/architecture.md) — then open a GitHub Issue if you're still stuck.

## Microsoft Support Policy

Support for Saturn is limited to the resources listed above.
